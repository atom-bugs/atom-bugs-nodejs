# Atom Bugs NodeJS package

NodeJS Debugging Plugin for [atom-bugs](https://github.com/atom-bugs/atom-bugs)

This Plugin uses the new `--inspect` v8-inspector (node) feature.

### Installation

```
apm install atom-bugs-nodejs
```

### Requirements
- atom 1.0.0+
- atom-bugs 1.0.0+
- NodeJS 6.3+
